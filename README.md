# ASR Benchmark
ASR benchmark comparing Deepgram, Sarvam, and Whisper on noisy Hindi/Hinglish audio for Indian locality name recognition.

## Models
- Deepgram Nova-2 (baseline)
- Sarvam Saaras V3
- Whisper medium

## Setup
1. Enter DEEPGRAM_API_KEY and SARVAM_API_KEY 
2. Upload audio_recording.zip to Colab
3. Run all cells top to bottom

## Requirements
pip install -r requirements.txt
